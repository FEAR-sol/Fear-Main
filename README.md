# FEAR Website

Creative studio that designs, develops, and deploys digital solutions.

## Tech Stack
- React (JavaScript)
- Tailwind CSS
- Framer Motion
- React Router

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Opens on http://localhost:3000

## Build

```bash
npm run build
```

## Design Specifications

- Desktop viewport: 1200px
- Layout: Relative positioning
- Fonts: Playfair Display, Inter, Josefin Slab
- Color scheme: Beige/cream background (#F5F3F0), dark accents (#1A1A1A)
